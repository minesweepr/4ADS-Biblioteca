# AP2 - Biblioteca Acadêmica Python
Atividade avaliativa da matéria Tópicos em ADS (4ADS)
## Instalação

Antes de prosseguir, verifique se possui o Python instalado.

```
# Se o ambiente virtual ainda não foi criado, utilize o seguinte comando
python -m venv .venv

# Ativar o ambiente virtual
.venv\Scripts\Activate

# Instalar todas as dependências
pip install -r requirements.txt

# Iniciar a aplicação
python main.py
```
